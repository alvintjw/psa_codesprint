# Getting Start

1. Clone the Repository
   First, clone the repository to your local machine:
   `git clone <repository-url>`

2. Set up backend

```bash
 cd backend
 npm install
 npm start dev
```

3. Set up frontend

```bash
 cd backend
 npm install
 npm start dev
```
